Instructions

This is exercise is quite challenging! Your goal is to create a simple drawing application that allows the user to pick the colour and size of their brush. They can also clear the screen to start a new drawing. The video below shows a complete application that has a little extra polish to make the application easier to use (the text labels for the controls and the brush preview). You can choose to focus on just the main functionality if you prefer.

Create an 800 by 600 canvas and add the three input controls (don't worry about the functionality just yet... simply put the controls on the screen):

  • A colour picker to select the brush colour.
  • A slider to set the brush size.
  • A button to clear the canvas.

You can put the controls wherever you like--your layout doesn't have to match the video example. To create the slider, consult the documentation for createSlider(). The minimum value should be the smallest size brush in pixels (I used 5). The maximum value should be the largest possible size for the brush (I used 50). The value should be the default size you'd like to use for the brush, which must be within the minimum and maximum values.

The drawing effect is created by drawing a circle at the mouse coordinates when the mouse is pressed. The fill colour of the circle should be the current value of the colour picker, and the diameter of the circle should be the current value of the slider. Getting the value of a slider is exactly the same as getting the value of a colour picker.

Remember there are hints available below the video and you can use check-exercise1-3.html to check your work.

Expected output:

The examples from lecture show how to use a slider as well as the other controls.

Where you call background() is important for this exercise. Because you want every circle placed when the user is drawing to stay on the screen, you should NOT call background() in draw(). Instead, call it in setup() to colour the background when the sketch loads. To clear the canvas, call background() when the clear button is clicked. This should happen in the event handler you create for the button.

To draw when the mouse is pressed, use the mouseIsPressed system variable in draw(). This is simpler and will create a smoother effect than using the mousePressed() event function.
